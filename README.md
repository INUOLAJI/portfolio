## 📂 Project Architecture & Pages

The application is structured into distinct, high-performance sections:

### 🏠 Home / Hero
Introduces who I am as a Full-Stack Developer with a clear, engaging primary call-to-action directing visitors straight to my active development builds. Features a customized framed profile photo layout.

### 💼 Projects Showcase
A gallery displaying live production platforms and open-source applications. Each project card includes descriptive copy, modular custom capability badges, and direct links to live software demos, source repositories, or individual deep-dive analysis pages.

### 📑 Project Case Studies (`/projects/:projectId`)
A dynamic routing page that automatically unpacks extensive technical blueprints for individual builds. It explicitly outlines the core **Business Challenge**, the developed **Solution**, full **Tech Stack breakdowns**, and the quantified **Business Impact**.

### 💡 About Me
Details my professional design philosophy of building tools that streamline operations and connect brands with target consumers. It houses my central interactive technical toolbox (React, Node.js, PHP, MongoDB, PostgreSQL, Python, etc.).

### ✉️ Get In Touch (Contact)
A fully optimized structural validation form for business inquiries, coupled with responsive shortcut access panels linking directly to my professional channels (LinkedIn, GitHub, and Twitter/X).
